---
这是一个批量生成文件夹的小程序
---
开发此程序的初衷是我每天都要上传拍摄好的照片在公司的网盘里，要新建很多文件夹和子文件夹，虽然有快捷键可以用，但是一次只能建一个文件夹，如果能一次性建很多文件夹那真的是泰酷辣！
于是这个小工具就诞生喽，嘿嘿。
